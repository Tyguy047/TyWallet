Get Started in Minutes

1

Clone Repository

Download TyWallet from GitHub and navigate to the project directory.

git clone https://github.com/Tyguy047/TyWallet
cd TyWallet
2

Install Dependencies

Install required Python packages using pip and the requirements file.

pip install -r requirements.txt
3

Launch TyWallet

Run the main application and start managing your cryptocurrency wallets!

python main.py

System Requirements

TyWallet works on Windows, macOS, and Linux. Requires Python 3.7+ and approximately 100MB of disk space. All dependencies are automatically installed via pip.
